Política de Privacidad - App Ischigualasto

Última actualización: 15 de diciembre de 2024

1. Información que Recopilamos

La aplicación Ischigualasto funciona como una interfaz para acceder al sitio web oficial del Parque Provincial Ischigualasto (www.ischigualasto.gob.ar). Recopilamos la siguiente información:

  • Información de uso básica para mejorar la experiencia de la aplicación
  • Datos de caché temporal para funcionamiento offline

2. Uso de la Información

La información recopilada se utiliza exclusivamente para:

  • Mejorar la funcionalidad de la aplicación
  • Permitir el acceso offline a contenido previamente visitado
  • Proporcionar una mejor experiencia de usuario

3. Permisos de la Aplicación

La aplicación requiere los siguientes permisos:

  • Acceso a Internet: Para acceder al contenido web
  • Almacenamiento: Para guardar contenido para uso offline

4. Contacto

Para cualquier consulta sobre esta política de privacidad, puede contactarnos a través del sitio oficial del Parque Provincial Ischigualasto.
